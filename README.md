# SaaS Idea Validator

SEO-focused Next.js marketing site and MVP validation tool for founders exploring SaaS ideas.

## Stack

- Next.js App Router
- TypeScript
- Tailwind CSS
- Local content system for blog posts, examples, FAQs, and programmatic pages
- JSON-LD schema helpers

## Local development

```bash
npm install
npm run dev
```

## Production checks

```bash
npm run lint
npm run build
```
